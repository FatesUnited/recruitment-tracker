About the Recruitment Tracker

The Vapor Lock Recruitment Tracker is an internal tool designed to assist recruiters in managing the recruitment and membership lifecycle of pilots within the alliance.

The system allows recruiters to track applicants, recruits, active members, and historical records of former pilots. Each pilot profile contains recruitment notes, interview information, onboarding status, and collaboration through recruiter comments.

This application helps ensure recruitment decisions are documented, organized, and easily accessible by the recruiting team.

About the Developer

This application was developed by Tribesmen.

The goal of this project is to create a clean, practical recruitment management tool tailored to the needs of alliance recruiters operating in New Eden.

Built using Django, PostgreSQL, and the official EVE ESI API for character data.
